Translation Engines

ScreenTranslate uses Apple Translation by default — completely free, private, and offline. For more options, bring your own API key.

Default

Apple Translation

Built into macOS. All translation happens on your device — no internet required after downloading language packs. No API key needed, no usage limits, completely free.

Free Private Offline No setup required

Bring Your Own Key (BYOK)

Optionally use a third-party translation engine by entering your own API key in Settings. Each service requires a separate account — ScreenTranslate does not provide or resell API keys.

DeepL

Known for natural, high-quality translations, especially for European languages. Offers a free tier with 500,000 characters per month.

Free: 500K chars/month Pro: from $5.49/month

Setup

  1. Create a free account at deepl.com/pro-api
  2. Copy your API key from the account dashboard
  3. In ScreenTranslate Settings, select "DeepL" and paste your key

Google Cloud Translation

Google's cloud-based translation service supporting 100+ languages. Offers 500,000 characters free per month with a Google Cloud account.

Free: 500K chars/month $20 per 1M chars

Setup

  1. Go to Google Cloud Console and create a project
  2. Enable the Cloud Translation API and create an API key
  3. In ScreenTranslate Settings, select "Google Cloud" and paste your key

Microsoft Azure Translator

Microsoft's cloud translation service with a generous free tier of 2 million characters per month. Supports 130+ languages.

Free: 2M chars/month $10 per 1M chars

Setup

  1. Go to Azure Portal and create a Translator resource
  2. Copy the API key and region from "Keys and Endpoint"
  3. In ScreenTranslate Settings, select "Microsoft Azure" and enter your key and region
